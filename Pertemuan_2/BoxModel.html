The CSS Box Model

Kotak di bawah ini menunjukkan struktur Box Model:

Ini adalah Konten (Width: 320px)
Padding: 10px (kiri+kanan = 20px)
Border: 5px (kiri+kanan = 10px)
Margin: 20px (di luar garis abu-abu)

Total Lebar = 320px + 20px + 10px = 350px.

Perhatikan bahwa margin memberi jarak antara kotak biru dengan teks ini.
